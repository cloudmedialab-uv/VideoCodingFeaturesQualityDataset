# Dataset statistical analysis and plots

> :exclamation: No funciona si el repositorio es privado (como lo es ahora mismo)

Characteristics such as mean, median, standard deviation, and visual representations of value distributions were obtained from the datasets with the TensorFlow Data Validation (TFDV), a tool that provides descriptive statistics and detects anomalities in the data.
